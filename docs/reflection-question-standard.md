# Reflection Question Standard

## Rule

For each lab in this series, ask exactly seven reflection questions.

The learner answers the questions. ChatGPT then uses the answers to prepare the final lab documentation and upload the result to GitHub.

## Standard Seven Questions

1. What was the main concept practised in this lab?
2. What part of the lab was most difficult or confusing?
3. How does your solution work, step by step?
4. What broke, failed, or needed fixing while you worked through it?
5. How does this lab connect to SRE, automation, or software engineering work?
6. What would need to change before this could be used in a real team or production environment?
7. What would you do differently next time?

## Documentation Handling

The learner should not need to write full lab documentation manually.

ChatGPT should use the learner's answers, notes, outputs, and errors to produce the final structured lab write-up.
